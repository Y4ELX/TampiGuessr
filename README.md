# 🗺️ ¿Cómo funciona?

1️⃣ Apareces en una **ubicación aleatoria de Tampico**

2️⃣ Puedes explorar usando **Street View**

3️⃣ Busca pistas como:

- letreros
- negocios
- arquitectura
- referencias locales

4️⃣ Marca en el mapa dónde crees que estás

5️⃣ Mientras más cerca estés, **más puntos ganas**
