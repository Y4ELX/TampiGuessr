# 🦀 TampiGuessr

**TampiGuessr** es un juego inspirado en *GeoGuessr* donde debes adivinar ubicaciones pero **solo dentro de Tampico, Tamaulipas**.

Explora calles, avenidas, colonias y rincones icónicos del sur de Tamaulipas mientras pones a prueba qué tan **tampiqueño eres realmente**.

Este es un **proyecto independiente hecho por un desarrollador de Tampico** con la intención de crear algo divertido para la comunidad local.

🌎 Juega aquí:
https://tampiguessr-17a8b.web.app/

<br>

# 🎮 ¿Cómo funciona?

1. Apareces en un punto aleatorio de **Tampico**.
2. Puedes moverte por el entorno usando **Street View**.
3. Analiza pistas como:

   * letreros
   * negocios
   * arquitectura
   * referencias locales
4. Marca en el mapa dónde crees que estás.
5. ¡Mientras más cerca estés, más puntos ganas!
<br>
# 🗺️ Características

* 🌎 Ubicaciones dentro de **Tampico, Tamaulipas**
* 📍 Sistema de puntuación basado en distancia
* 🧭 Navegación libre con Street View
* 🎯 Mecánica simple pero desafiante
* 🇲🇽 Experiencia centrada en una ciudad mexicana
* 🦝 Referencias locales para verdaderos tampiqueños


<br>

# 🧠 ¿Qué tan tampiqueño eres?

Este juego responde la pregunta más importante:

> **¿Realmente conoces Tampico o solo sabes llegar a Altama?**

Pon a prueba tu conocimiento de:

* colonias
* avenidas
* plazas
* lugares icónicos
* calles perdidas que solo los locales reconocen


<br>

Este proyecto utiliza varias tecnologías web modernas:

* **TypeScript**
* **Angular**
* **Google Maps / Street View API**
* **Frontend Web**
* **Firebase Hosting**
